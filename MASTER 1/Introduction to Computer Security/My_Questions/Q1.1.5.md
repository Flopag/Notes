# Q1.1.5

> Give summation, multiplication and power calculus simplifications

## Summation

(x+y) % n = (x % n + y % n) % n

ex :

- 27 % 2 = (20 % 2 + 7 % 2) % 2 = (0 + 1) % 2 = 1

## Multiplication

(x . y) % n = (x % n . y % n) % n

ex :

- 28 % 2 = (4 % 2 . 7 % 2) % 2 = (0 . 1) % 2 = 0
