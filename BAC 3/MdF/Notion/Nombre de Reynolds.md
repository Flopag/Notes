# Nombre de Reynolds

- Symbole : $Re$
- Dimension : /
- Formule : $Re = \frac{U_c \ L_c}{\nu_c}$ avec $\nu_c = \frac{\mu_c}{\rho_c}$
- Signification : Caractérise la nature du régime d'écoulement

$$\left\{
     \begin{array}{ll}
		Laminaire \ - \ Re < 2000\\
	     Transitoire \ - \ 2000 < Re < 3000\\
	     Turbulent \ - \ Re > 3000
     \end{array}
\right.$$
